# Useful Resources

## Overview

This file contains curated external resources to support enumeration, exploitation, privilege escalation, and Active Directory attacks.

These are **reference augmentations**, not replacements for this field guide.

Use them when:
- deeper research is needed
- a technique is unfamiliar
- you need payload variations or edge cases
