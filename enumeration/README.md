# Enumeration

## Purpose

Systematically identify:
- services
- users
- credentials
- attack surface

---

## Critical Concept

Enumeration is NOT a phase.

It is:
→ continuous

---

## When To Enumerate

- initial access
- after shell
- after privilege escalation
- after pivoting
